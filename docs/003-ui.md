# Spec 003: 로그인 화면

## 개요

Google 로그인 화면에 대응하는 mock-google-oidc의 유일한 사용자 대면 화면.
`GET /o/oauth2/v2/auth`에서 렌더링된다.

## 화면

```
+----------------------------------------------------------+
|  Test IDP                                                |
|----------------------------------------------------------|
|                                                          |
|  Sign in to continue                                     |
|                                                          |
|  Email : [ alice@example.com                          ]  |
|  Name  : [ Alice                                      ]  |
|                                                          |
|                    [ Login ]                              |
|                                                          |
|  ▸ Advanced (Response Mode)                              |
|    (•) Normal   ( ) Deny   ( ) Token Error               |
|    ( ) Userinfo Error                                    |
|                                                          |
|  ── Debug Info ──────────────────────────────────────    |
|  client_id: my-app                                       |
|  redirect_uri: http://localhost:8080/callback             |
|  scope: openid email profile                             |
|  state: abc123                                           |
+----------------------------------------------------------+
```

## 구성 요소

### 1. Identity 입력 (핵심)

| 필드 | 기본값 | 설명 |
|------|--------|------|
| Email | `alice@example.com` (또는 `login_hint` 값) | 유저 이메일 |
| Name | `Alice` | 유저 이름 |

기본값이 채워져 있어서 바로 Login만 눌러도 동작.

### 2. Login 버튼

폼을 `POST /o/oauth2/v2/auth`로 제출. 이것만 누르면 로그인 완료.

### 3. Advanced — Response Mode (접힘)

기본 접혀있음. 에러 테스트 시에만 펼침.

| 값 | 설명 |
|----|------|
| `normal` | 정상 로그인 (기본) |
| `deny` | access_denied 반환 |
| `token_error` | /token에서 500 |
| `userinfo_error` | /userinfo에서 500 |

### 4. Debug Info (읽기 전용)

OAuth 파라미터 표시. 디버깅 편의용.

### Hidden Fields

```html
<input type="hidden" name="redirect_uri" value="...">
<input type="hidden" name="state" value="...">
<input type="hidden" name="nonce" value="...">
<input type="hidden" name="scope" value="...">
<input type="hidden" name="client_id" value="...">
<input type="hidden" name="code_challenge" value="...">
<input type="hidden" name="code_challenge_method" value="...">
```

PKCE 파라미터도 hidden field로 전달되어 code 생성 시 저장된다.
